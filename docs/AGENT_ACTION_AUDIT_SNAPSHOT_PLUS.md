## Call structure

Suggested 45-minute structure:

1. 5 min — workflow context
2. 10 min — action boundary and current permission model
3. 10 min — first trust break and missing evidence
4. 10 min — ALLOW / BLOCK / ESCALATE recommendation
5. 10 min — hardening roadmap and next step

## Hardening roadmap shape

The roadmap should be short and prioritized:

| Priority | Control | Why it matters |
| --- | --- | --- |
| P0 | Must fix before scaling | Blocks high-risk trust break |
| P1 | Fix soon | Reduces ambiguity or replay gaps |
| P2 | Improve later | Makes governance and review easier |
